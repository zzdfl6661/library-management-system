# 图书馆管理系统修复计划（第二版）

## 问题分析

### 问题1：借书证管理功能中新版借书证输入框不能正常输入
- **现状**：输入框显示但无法输入
- **原因**：可能是样式覆盖或Element Plus版本问题

### 问题2：借书管理功能中可借数量为空
- **现状**：输入借书证号后显示"可借数量：本"
- **原因**：后端返回的`LibraryCard`实体没有`availableCount`字段

### 问题3：学生借阅后我的借阅没有数据
- **现状**：学生借阅图书后，"我的借阅"页面显示No Data
- **原因**：`MyBorrows.vue`使用`username`查询，但后端API需要学生学号或ID

## 修复方案

### 1. 修复借书证输入框问题
- 添加必要的CSS样式确保输入框可交互

### 2. 添加可借数量字段
- 创建`LibraryCardResponse` DTO
- 修改`CardController`返回带`availableCount`的响应

### 3. 修复学生借阅记录查询
- 修改`MyBorrows.vue`使用正确的查询方式

## 文件修改清单

| 文件路径 | 修改内容 |
|---------|---------|
| `CardManagement.vue` | 修复输入框样式 |
| `CardController.java` | 返回带可用数量的响应 |
| `LibraryCardServiceImpl.java` | 添加可用数量计算 |
| `MyBorrows.vue` | 修复借阅记录查询逻辑 |
